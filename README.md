# Travel & Tourism Website

This is a responsive **Travel and Tourism Website** built using **HTML, CSS, and JavaScript**.  
The website provides information about popular tourist destinations and famous landmarks.

## About the Project

The project contains multiple pages showcasing different locations such as:
- Paris
- Marseille
- Rennes
- Eiffel Tower
- Opera House
- Arc de Triomphe

It is designed with a clean layout, images, icons, and smooth navigation to give users a good browsing experience.

This project was created to improve frontend development skills and understand real-world website structure.

## Features

- Responsive design for all devices
- Multiple location-based pages
- Image galleries and SVG icons
- Search location page
- Contact page
- Privacy policy page
- Clean folder structure

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- GitHub Pages (for hosting)
